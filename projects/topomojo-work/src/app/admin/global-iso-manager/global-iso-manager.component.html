Global Files

Upload files to the global library to make them available across all workspaces. If an uploaded file is not in ISO format, TopoMojo will wrap the file in an ISO after uploading. ISOs can be attached to VM templates via the template settings or dynamically per-variant via the variant settings on the Challenge tab.
Note: Large files (>1GB) may take additional time to process after upload completes. The list will automatically refresh when uploads finish.
